African Art Gallery

This is a 3D web app for an African Art Gallery created using Vanilla HTML, CSS, Javascript, Three.Js, J Query and Node.Js

This is my Master's project which would involve users exploring an art gallery specifically designed to foster and show African art from artists which would also include good user input and user experience practices sufficiently engaging the users as they tour the gallery.
